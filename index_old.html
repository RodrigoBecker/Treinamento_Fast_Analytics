O que é Projeto?

É o contexto no qual será inserido todas as informações do cliente na plataforma.

Como assim?

O que é Patch BaInstaller

Para gerar as querys com os dados dos moódulos do protheus é necessário que seja configurado um patch que irá gerar essas consultar nos modulos e assim montar os dados para serem extraídos.

O que é o Agent

O Agent é um arquivo utilizado como extrator de dados, sua função é acessar as querys geradas pelo BAInstaller extrair os dados e enviá-los para o projeto.
